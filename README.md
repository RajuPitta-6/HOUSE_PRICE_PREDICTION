🏠 House Price Prediction System

An end-to-end Machine Learning application that predicts house prices based on property and location features.

🚀 Features

- House price prediction using Machine Learning
- FastAPI backend
- HTML, CSS, JavaScript frontend
- Real-time predictions
- Model comparison and selection
- Preprocessing pipeline

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- FastAPI
- HTML
- CSS
- JavaScript

📊 Features Used

- City
- Locality Tier
- BHK
- Bathrooms
- Super Area (sqft)
- Property Age
- Parking
- Furnishing
- Distance to City Center

🤖 Models Trained

- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
